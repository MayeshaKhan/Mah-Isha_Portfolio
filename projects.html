JobNest

This project has similar features with Upwork, work marketplace. This application works in subscription basis. There subscription types- Basic, Standard, and Premium. Auth feature is common for all the users. So here, users can upload for jobs, see the jobs and apply for the desired job. Also, users can update their profile as required as their job. An AI feature is built here so the users can have the facility of AI advantages, this feature is available for the premium subscribers. Users can attend quizzes related to their skills, this feature is available for premium and standard subscribers. Job proposal, job feedback and helpline feature is available for all type of subscribers. This project was the part of “Advanced programming in Web Technology" course utilizing Next.js for the frontend and NestJS for the back-end.

SwiftCart

In this project, APIs are built with ASP.NET Framework based on SOLID principle and 3 tier architecture. There are several roles for admin, customer, agent, and seller. Auth feature is common for all the users. Customers can order product and read review, write review for the product. Every branch has an agent. Agent categorises the products from the seller. Seller sells products, also can order product. Admin has the ability of CRUD operation over the users, products and the categories. This project was a part of “Advanced programming with ASP.NET” course.

Bus ticketing management system

As part of my “Web Technogies” courses, this project was done with HTML, CSS, JavaScript, PHP. Is this project there are four users; admin, passengers, vendor, and accountants. Passenger can book ticket, get discount by using coupon, payment and cancel ticket. Passenger can see the available buses for trip and all discount coupons. While booking ticket passenger can see all the available seats. Accountants manages all the expenses like billing and invoicing, discount, refund and account history of passengers. Vendors manages the bus schedule for the trip such as updating bus schedule, cancel bus trip, allocating new bus trip. Also they can see the passengers’ list for a particular bus/trip.
